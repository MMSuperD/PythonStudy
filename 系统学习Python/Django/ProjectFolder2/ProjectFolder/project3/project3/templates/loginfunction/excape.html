html转义：
{{ content }}
使用过滤器safe 来转义
{{ content|safe }} 使用autoexcape关闭转义
{% autoescape on %} {{ content }} {% endautoescape %}
使用autoexcape打开转义
{% autoescape off %} {{ content }} {% endautoescape %} 如果是转义字符，默认转义
{% autoescape off %} {{ content1 }}
{{ content }}
{% endautoescape %}
模板硬编码中，字符串默认不会经过转义：
{{ test|default:'

I love you too

' }}
手动进行转义:
{{ test|default:'<h1> I love you too</h1>' }}
